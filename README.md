# Advent of Code 2024 - Samdesk.io Assessment

Solutions by Dixit Ghodadara

## Tech Stack

- TypeScript + Node.js
- Jest for testing
- ESLint + Prettier for code quality

## Setup

```bash
npm install
```

## Commands

| Command | Description |
|---------|-------------|
| `npm test` | Run all tests |
| `npm run test:coverage` | Run tests with coverage |
| `npm run build` | Compile TypeScript |
| `npm run solve:day02` | Run Day 02 solution |
| `npm run lint` | Lint source files |
| `npm run format` | Format source files |
